## List drives
```
user@host:/ $ lsblk
NAME        MAJ:MIN RM   SIZE RO TYPE MOUNTPOINTS
sda           8:0    0 111.8G  0 disk 
└─sda1        8:1    0 111.8G  0 part /mnt/audiobooks
mmcblk0     179:0    0  29.7G  0 disk 
├─mmcblk0p1 179:1    0   512M  0 part /boot/firmware
└─mmcblk0p2 179:2    0  29.2G  0 part /
```

## Mount to location
```
user@host:/dev $ sudo mkdir /mnt/mymountpoint
user@host:/dev $ sudo mount /dev/sda1 /mnt/mymountpoint
user@host:/dev $ cd /mnt/mymountpoint
```
